# Proyecto Git

Proyecto de práctica para el examen final de Git y GitHub.

## Funcionalidades
- Estructura base HTML
- Barra de navegación simulada en el encabezado

## Ramas utilizadas
- main: rama principal del proyecto
- feature/navbar: rama para el desarrollo de la barra de navegación

## Flujo de trabajo
Se utilizó Git para el control de versiones y GitHub como repositorio remoto,
simulando un entorno de trabajo colaborativo mediante ramas y merge.
